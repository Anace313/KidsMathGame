    ### Welcome to Kids Math Game This is a Demo Game created by Html5 / Css3 / Javascript / angularJs. With this game kids will test their addition, subtraction, multiplication and division skills.

    This is a demo version of the game so it only contains small number of questions.

    Before Testing this game in your browser you should run this code into terminal

       ` $ cd /kidsMathGame_folder`
    `$ npm install`
    `$ bower install`
